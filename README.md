# 2026-pds--ChamaFe-
**ChamaFé** é uma plataforma web de gestão paroquial desenvolvida para centralizar e automatizar a administração de acampamentos, eventos, células, voluntários e participantes. O sistema visa reduzir processos manuais, melhorar a organização, facilitar a comunicação e fornecer relatórios estratégicos para a coordenação pastoral.
